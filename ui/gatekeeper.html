First time on this Mac?

macOS may block Beacon once because it isn’t from the App Store. That’s normal — you only need to do this once.

1Open the download

Double-click the Beacon file you received (the DMG).

2Install Beacon

Double-click Beacon — it copies itself to Applications.

3If Mac says it can’t be opened

Open Applications → right-click Beacon → Open → click Open again.

Then open Beacon anytime: ⌘Space → type Beacon → Enter.
