Beacon

First time on this Mac?

macOS may block Beacon once because it isn’t from the App Store. That’s normal — this only happens the first time.

  1. 1
    Open the download

    Double-click the Beacon file you received (the DMG).

  2. 2
    Install Beacon

    Double-click Beacon in the window — it copies itself to Applications.

  3. 3
    If Mac says it “can’t be opened”

    Open Applications → right-click Beacon → Open → click Open again.

After that, open Beacon anytime with ⌘Space → type Beacon → Enter.
